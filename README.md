# budget-cli

Simple command-line tool to track monthly expenses and set savings goals.

## Features

- Add income and expense entries
- - Set monthly budget limits per category
  - - Export summary to CSV
    - - Savings goal tracker with progress
     
      - ## Install
     
      - ```bash
        pip install budget-cli
        ```

        ## Quick start

        ```bash
        budget add expense food 45.50
        budget add income salary 3200
        budget summary --month 2026-06
        ```
